# OCR 區域擷取工具

一個純前端、單檔的網頁工具,用來在截圖或圖片上框選特定區域,並輸出該區域的座標(`left / top / width / height`),方便用於 OCR 比對、自動化腳本設定等用途。

## 🚀 線上使用

直接開啟即可,無需安裝:

**👉 https://deantw69.github.io/ocr-region-tool/**

## ✨ 功能

- **三種載入方式**
  - 點「開啟圖片」選檔
  - `Ctrl / Cmd + V` 直接貼上截圖
  - 把圖片檔拖進畫面
- **框選工具**:在圖上拖曳建立選取框,即時顯示範圍座標
- **座標以原始圖片像素為基準**:不受畫面縮放影響,可直接拿去做 OCR 比對
- **多種微調方式**
  - 拖曳 8 個控制點縮放、拖框內部移動
  - 右側欄位直接輸入數值即時更新
  - 方向鍵移動 1px、`Shift` + 方向鍵改大小、`Alt` + 方向鍵一次 10px、`Esc` 取消
- **JSON 輸出**:格式為 `{ "left": 859, "top": 714, "width": 202, "height": 34 }`,可一鍵複製 JSON 或數值
- **貼上 JSON 跳至範圍**:貼上座標 JSON 即可把框選跳到指定位置(支援無大括號的片段)
- **框選區域另存新檔**
  - 支援系統「另存新檔」對話框(Chrome / Edge),可自選位置與檔名
  - 可選 PNG / JPEG 格式
  - 可自訂檔名前綴、自由勾選是否加入座標後綴,並即時預覽檔名
- **載入新圖時保留框選位置**(可勾選):方便連續對多張圖比對同一區域
- **縮放檢視**:`Ctrl / Cmd + 滾輪` 縮放、符合視窗、1:1 等

## 🖱️ 操作說明

| 操作 | 說明 |
| --- | --- |
| 拖曳 | 建立框選 |
| 拖框內部 | 移動框選 |
| 拖 8 個控制點 | 縮放框選 |
| 方向鍵 | 移動 1px |
| `Shift` + 方向鍵 | 縮放 |
| `Alt` + 方向鍵 | 一次移動 / 縮放 10px |
| `Esc` | 取消框選 |
| `Ctrl / Cmd + V` | 貼上截圖 |
| `Ctrl / Cmd + 滾輪` | 縮放檢視 |

## 💻 本機執行

本工具是單一 `index.html`,無任何相依套件。

直接以瀏覽器開啟 `index.html` 即可使用。若需要剪貼簿讀取與「另存新檔」對話框等需要 secure context 的功能,建議透過本機伺服器執行:

```bash
python3 -m http.server 8765
# 接著開啟 http://127.0.0.1:8765/index.html
```

git add -A

git push
